About This Project

The Global Oil & Gas Infrastructure Atlas maps the world's energy systems through interactive visualizations. Each map uses Leaflet.js with animated pipeline flows, pulsing markers, and detailed data popups covering production rates, reserves, operational status, and geopolitical context.

Interactive

Zoom, pan, click. Hover pipelines for instant data. Filter by category or region.

Real Data

Production rates, capacities, reserve estimates. Status: active, damaged, shut, proposed.

Geopolitics

Sanctions, conflicts, guerrilla attacks, referendum shutdowns, new discoveries.

Self-Contained

Single HTML files. No build tools. Works offline. Open in any browser.

Data note — Production figures and reserve estimates are based on publicly available data as of early 2026. Values are approximate, intended for visualization. Pipeline routes are simplified.
