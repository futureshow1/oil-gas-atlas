About This Project

The Global Oil & Gas Infrastructure Atlas is an open-source data visualization project mapping the world's energy infrastructure. Each interactive map uses Leaflet.js for rendering with a dark CartoDB base layer, featuring animated pipeline flows, pulsing field markers, and detailed popups with production data, capacity, reserves, and operational status.

Interactive Maps

Zoom, pan, click any element for detailed popups. Hover pipelines for instant info. Filter by category — pipelines, fields, or LNG.

Real Data

Production rates, pipeline capacities, reserve estimates, commissioning years. Status tracking: active, damaged, shut, proposed, under construction.

Geopolitical Context

Sanctions impacts, conflict zones, guerrilla attacks on pipelines, referendum-ordered shutdowns, new discoveries transforming economies.

Self-Contained

Each map is a single HTML file — no build tools, no dependencies beyond CDN. Open in any browser. Works offline after first load.

Data disclaimer: Production figures, reserve estimates, and status information are based on publicly available data as of early 2026. These are approximate values for visualization purposes. Actual figures may vary. Pipeline routes are simplified representations.
